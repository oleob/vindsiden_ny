# Vindsiden.no alternative frontend
Uses the API-provided by [Vindsiden.no](http://vindsiden.no/)

## Demo
Click [here](https://boring-dubinsky-ac8af1.netlify.com/) to view the site in action.

## How to run
1. Clone repository
2. Run npm install
3. Run npm start

## Libraries used
* React
* Redux
* Material-ui
* Chartkick
